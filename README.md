# README

Ruby version: 2.4.6

# Run project:

```
docker-compose up
```

# Acess project on:

```
http://localhost:3000
```

# Acess API docs on:

```
http://localhost:3000/api-docs/index.html
```

# Run console:

```
docker-compose run app bash
```

# Run Rspec:

```
bundle exec rspec
```

# Generate swagger docs:

```
bundle exec rails swagger
```
